# BichoBet Landing Page

Landing page institucional premium desenvolvida com Next.js 15, React, TypeScript, Tailwind CSS, componentes no estilo shadcn/ui e Framer Motion.

## Rodando o projeto

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm start
```

## Observações

- O projeto entrega uma experiência institucional e uma lista de espera funcional no navegador.
- Não inclui fluxos reais de apostas, pagamentos, KYC ou operações de gambling.
